Cryptage Asymétrique

Le cryptage asymétrique, connu sous le nom de cryptage à clé publique, utilise des paires de clé publique et de clé privée mathématiquement liées pour crypter et décrypter les données sensibles de l'expéditeur et du destinataire.

Chiffrement Symétrique

Le chiffrement symétrique est un type de chiffrement dans lequel une seule clé (une clé secrète) est utilisée pour chiffrer et déchiffrer les informations électroniques.

Les entités communiquant via un chiffrement symétrique doivent échanger la clé afin qu'elle puisse être utilisée dans le processus de déchiffrement.

C'est le plus ancien type de méthode de cryptographie utilisé dans l'histoire humaine, 1900 avant JC pour être précis dans l'Ancien Empire d'Égypte sur des hiéroglyphes non standard gravés dans le mur.

Le concept de base est qu'il y a des cas où nous voulons nous assurer qu'un message donné est lu exclusivement par l'expéditeur et le destinataire.

Définitions

PLAINTEXT : le message clair avant de le chiffrer.

CIPHERTEXT : le message qui a été chiffré.

CHIFFREMENT : processus de cryptage d'un message donné de manière que seules les parties autorisées puissent y accéder.

DÉCRYPTAGE : processus de décryptage d'un texte chiffré donné.

CLEF : il s'agit d'une séquence nécessaire à la fois au chiffrement et au déchiffrement.
